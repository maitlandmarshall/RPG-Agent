
# Visual Style: Planet Vegeta

**Reference Style**: *Dragon Ball Super: Broly (Movie)*

**Key Elements**:
- **Line Art**: Thick, rough, dynamic line weights. "Shintani Style".
- **Colors**: Vibrant, saturated, but with a gritty filtering. High contrast.
- **Lighting**: Dramatic, movie-quality lighting.
- **Atmosphere**: Ancient, brutal, alien.
- **Keywords for Generation**: `DB Super Broly style`, `thick line art`, `90s aesthetic modern rendering`, `vibrant colors`, `gritty texture`, `Akira Toriyama artstyle`.

**Character Originality Rules**:
- **NO CLONES**: Do not generate characters that resemble ANY existing Dragon Ball character (Canon or Non-Canon).
- **Process**: First write a detailed physical description (Face shape, eyes, scars, hair texture, body type, unique marks). Use this description to generate the image.
- **Bespoke Designs**: Armor must be scavenged/custom. No standard uniforms.
